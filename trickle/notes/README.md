# بديلي (Badeeli)

منصة ويب خفيفة وسريعة (SPA) مخصصة للفنيين الصحيين في المملكة العربية السعودية لتبادل أماكن العمل بين المناطق المختلفة.

## الميزات الأساسية:
- تصميم مدمج وأنيق (متوافق تماماً مع الجوال يشبه تطبيقات iOS).
- عرض مباشر للطلبات بدون شاشات ترحيبية.
- فلترة ذكية حسب التخصص والمناطق.
- إضافة طلب جديد بخطوات مبسطة.
- حفظ حالة المستخدم محلياً (localStorage) لمنع تكرار الطلبات.
- التكامل مع قاعدة بيانات Trickle لجلب وحفظ الطلبات (جدول `TransferRequest`).
- وجود لوحة تحكم للإدارة في المسار `/adminn.html` تتيح استعراض، فلترة، وحذف الطلبات مع إحصائيات مبسطة.

## ملاحظة للمطور الذكاء الاصطناعي:
عند تحديث المشروع، تأكد من مراجعة هذا الملف وتحديثه إذا تمت إضافة ميزات رئيسية جديدة.
